E-Commerce Dashboard

October 2024 3 weeks Individual Featured Project
HTML5 CSS3 Grid Bootstrap 5 JavaScript Chart.js Responsive Design
Problem Statement

Small business owners need a centralised interface to monitor sales, manage inventory, and track customer orders without relying on expensive third-party SaaS solutions.

Solution

A responsive admin dashboard built entirely with front-end technologies, featuring data visualisation, product management, and order tracking with local storage persistence.

Key Features

  • Responsive sidebar navigation with collapsible menu
  • Interactive sales charts using Chart.js
  • Product CRUD operations with form validation
  • Order management with status tracking
  • Real-time statistics cards with animated counters
  • Data tables with sorting and search functionality
  • Dark/light mode theme toggle
  • Fully responsive across all breakpoints
  • localStorage persistence for data
  • Accessibility-compliant semantic HTML

What I Learned

This project deepened my understanding of CSS Grid for complex dashboard layouts, introduced me to data visualisation with Chart.js, and reinforced the importance of component-based thinking in JavaScript. I also gained practical experience with CRUD operations and state management using localStorage.

Task Manager Application

November 2024 2 weeks Individual Kanban Style
HTML5 CSS3 Flexbox Bootstrap 5 JavaScript ES6+ Drag & Drop API localStorage

Individuals and small teams struggle to organise tasks effectively, track progress visually, and maintain productivity without complex project management tools.

Solution

A Kanban-style task manager with drag-and-drop functionality, task categorisation, priority levels, due dates, and local storage persistence for offline use.

Key Features

  • Drag-and-drop task cards between columns
  • Create, edit, and delete tasks inline
  • Priority levels with colour coding
  • Due date assignment with visual indicators
  • Task filtering by priority and status
  • Search functionality across all tasks
  • Task counter per column
  • Progress overview dashboard
  • localStorage auto-save
  • Mobile-optimised touch interactions

What I Learned

Building this task manager taught me how to implement the HTML5 Drag and Drop API, manage application state in JavaScript, and create complex interactive UIs. I also learned about event delegation for dynamically created elements and the importance of UX considerations like visual feedback during drag operations.

Weather Dashboard

December 2024 2 weeks Individual API Integration
HTML5 CSS3 Bootstrap 5 JavaScript Fetch API Geolocation API OpenWeatherMap API
Problem Statement

Weather information is often scattered across multiple sources or buried in cluttered interfaces. Users need a clean, personalised dashboard for quick weather checks and planning.

Solution

A single-page weather application that fetches real-time data from the OpenWeatherMap API, displays current conditions, 5-day forecasts, and temperature trends with geolocation support.

Key Features

  • Real-time weather data via Fetch API
  • Current conditions display with icons
  • 5-day forecast with temperature ranges
  • Temperature trend line chart
  • Geolocation-based auto-detection
  • City search functionality
  • Weather detail cards (humidity, wind, pressure)
  • Responsive card-based layout
  • Error handling for failed requests
  • Loading states with skeleton screens

What I Learned

This project was my deep dive into asynchronous JavaScript. I mastered the Fetch API, learned to handle API keys and rate limits, implemented error handling for network failures, and worked with JSON data structures. The geolocation integration also taught me about browser permissions and graceful fallbacks when APIs are unavailable.
