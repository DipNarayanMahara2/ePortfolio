FAQ Accordion (Event Delegation)

let declares a block-scoped variable that can be reassigned. const declares a block-scoped variable that cannot be reassigned (though objects and arrays declared with const can still be mutated). Both are preferable to var, which is function-scoped and hoistable.

The DOM (Document Object Model) is a tree-like representation of an HTML document. JavaScript can manipulate it using methods like document.getElementById(), document.querySelector(), and document.querySelectorAll() to select elements, then modify their content, styles, attributes, or structure.

Event delegation is a technique where a single event listener is attached to a parent element to handle events for all its children. It's efficient because you only need one listener instead of many, and it automatically works for dynamically added elements. Use event.target to identify which child triggered the event.

async/await is syntactic sugar over Promises. An async function always returns a Promise. Inside it, you can use await to pause execution until a Promise resolves, making asynchronous code read like synchronous code. This improves readability and error handling with try/catch blocks.

localStorage is a Web Storage API that stores key-value pairs persistently in the browser (until explicitly cleared). It's useful for saving user preferences, caching data, or storing application state. Data is limited to ~5MB per origin and only stores strings (use JSON.stringify/parse for objects). For session-only storage, use sessionStorage.
