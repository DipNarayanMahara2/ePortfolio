Cascading Style Sheets control the visual presentation of web pages. Explore selectors, the box model, layouts, animations, and responsive design through interactive examples.

Three Ways to Apply CSS

CSS can be applied inline, internally within a <style> tag, or externally via a linked stylesheet. External CSS is the recommended approach for maintainability.

Inline CSSstyle attr

This text is styled inline

<p style="color: green; font-weight: bold;">Inline styled text</p>

Internal CSS<style>

Styled via internal <style> tag

<style> .internal { color: blue; } </style> <p class="internal">Internal CSS</p>

External CSS.css file

Linked external stylesheet (this site uses this approach)

<link rel="stylesheet" href="css/styles.css"> /* In styles.css */ .external { color: purple; }

Control how elements are positioned in the document flow.

Position TypesLayout

static
relative
absolute
/* Static - default flow */ .static { position: static; } /* Relative - offset from normal */ .relative { position: relative; top: 10px; left: 10px; } /* Absolute - relative to ancestor */ .absolute { position: absolute; top: 10px; right: 10px; } /* Fixed - relative to viewport */ .fixed { position: fixed; bottom: 20px; } /* Sticky - hybrid behaviour */ .sticky { position: sticky; top: 0; }

Pseudo-elements::before

The first line of this paragraph is styled differently thanks to the ::first-line pseudo-element. It automatically adapts to the container width, styling only the text that appears on the first visual line of the element.

The first letter of this paragraph is styled as a drop cap using the ::first-letter pseudo-element, creating an elegant typographic effect commonly seen in print publications.

/* First line styling */ p::first-line { font-weight: bold; color: #2c5041; } /* Drop cap */ p::first-letter { font-size: 3em; float: left; margin-right: 8px; color: #2c5041; }
