# ePortfolio - Full Stack Web Development

## Overview

This is a comprehensive **Full Stack Web Development ePortfolio** created as a university assignment. It showcases practical skills in modern web technologies including HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5 framework. The portfolio demonstrates both technical knowledge through demo pages and real-world project implementations.

## Project Structure

### 📄 Pages

- **index.html** - Main landing page with an immersive hero section and portfolio overview
- **projects.html** - Detailed showcase of full-stack projects including:
  - E-Commerce Dashboard
  - Task Manager Application
  - Weather Dashboard
- **contact.html** - Contact form and communication information
- **html-demo.html** - HTML5 features and semantic markup demonstration
- **css-demo.html** - CSS3 capabilities including layouts, animations, and styling techniques
- **js-demo.html** - JavaScript functionality demonstrations and DOM manipulation
- **bootstrap-demo.html** - Bootstrap 5 component library and responsive design examples
- **fullstack-demo.html** - Integrated full-stack application examples

### 📁 Assets

- **css/styles.css** - Premium custom stylesheet with CSS variables, design tokens, and responsive layouts
- **js/main.js** - Core JavaScript functionality and event handling
- **images/** - Project images and visual assets

### 🚀 Project Showcases

## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required - runs as static HTML/CSS/JS

### How to Use

1. Open `index.html` in your web browser
2. Navigate through the different pages using the navigation menu
3. Explore demo pages to see technology implementations
4. Check out the projects section for detailed portfolio examples
5. Use the contact page to reach out
